JavaScript slice() Method Explained

Understanding negative indices in slice()

What is the slice() method?

The slice() method extracts a section of a string and returns it as a new string.

It takes two parameters: start and end (end is not included in the result).

Negative indices count backwards from the end of the string.

// In your code: const gameName = "chessBaseIndia"; // 14 characters const anotherString = gameName.slice(-8, 4); console.log(anotherString); // What will this output?

Key Points About slice()

  • slice() extracts a section of a string and returns a new string
  • Negative indices count backwards from the end of the string
  • slice(-n) means "start from the nth character from the end"
  • If start index is greater than end index, slice() returns an empty string
  • Unlike substring(), slice() accepts negative indices
// More examples: const str = "javascript"; console.log(str.slice(0, 4)); // "java" console.log(str.slice(4)); // "script" console.log(str.slice(-6)); // "script" (from index 5) console.log(str.slice(-6, 10)); // "script" console.log(str.slice(2, -3)); // "vasc" (from index 2 to index 7) console.log(str.slice(-3, -1)); // "pt" (from index 7 to index 9)

Understanding slice() with negative indices is useful for string manipulation!
